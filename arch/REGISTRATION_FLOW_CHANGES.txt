# SISTEM PENDAFTARAN BARU & RBAC PERGURUAN/ATLET

## CURRENT FLOW (TO CHANGE)
- Siapa saja bisa register user → auto role='coach' → langsung login
- Coach daftar athlete → pending → admin verify
- Athlete.coach_id → perguruan tidak formal

## NEW FLOW
```
1. PENDAFTARAN PERGURUAN/OFFICIAL
   ├─ Perguruan isi form /auth/register-perguruan
   │  (name, email, password, perguruan_name, address, phone)
   ├─ User dibuat: role='perguruan', status='pending'
   ├─ Email admin: \"New perguruan pending\"
   └─ Response: \"Pending admin approval\"

2. ADMIN VERIVIKASI
   ├─ Admin GET /admin/pending-perguruan
   ├─ PATCH /admin/perguruan/{id}/approve
   │  Update status='active', approved_by=admin.id
   └─ Email perguruan: login credentials

3. LOGIN & MANAGE ATLET
   ├─ Perguruan login
   ├─ POST /athletes → only own perguruan
   │  athlete.perguruan_id = user.id
   │  athlete.club = perguruan.name
   └─ EventParticipant: only athletes own perguruan

4. ATLET EVENT
   ├─ POST /events/{e}/participants
   ├─ status='pending'
   └─ Admin verify → 'verified'
```
